## Suggested reading order

1. Run through [Quick Start](/en/docs/quick-start) until `doctor`, `devices list`, and `test event` all behave normally.
2. Open [Configuration](/en/docs/configuration) and get the shape of `devices`, `actions`, and `profiles` straight.
3. If your main focus is the gesture button, continue with [Gestures](/en/docs/gestures).
4. If behavior still feels unstable on hardware, go straight to [Troubleshooting](/en/docs/troubleshooting).
